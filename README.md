# Analysis-urban-transport-conflict



Разработка подхода к анализу конфликтов, связанных со строительством объектов транспортной инфраструктуры

Выпускная квалификационная работа (магистерская диссертация)



Университет ИТМО, Институт дизайна и урбанистики

Направление: 07.04.04 Градостроительство

Образовательная программа: Цифровая урбанистика



Автор: Зубова З.А.

Научный руководитель: Шмелева И.А., к.п.н., доцент

Год: 2026



\## Аннотация

В работе разработан комплексный подход к анализу градостроительных конфликтов, возникающих при строительстве объектов транспортной инфраструктуры на природных территориях. Подход основан на комбинированном применении качественного анализа нормативно-правовой документации (с использованием инструментов ИИ) и экспертной оценки результатов по методу Дельфи.
